Key skills

Data cleaning and preprocessing:
Missing values and potential outliers were addressed, and numerical features were scaled. Wine type was encoded as a binary variable
Logistic regression:
RFE was used to select six key features for predicting wine type: chlorides, residual sugar, fixed acidity, volatile acidity, pH, and sulphates. The model achieved 97.3% accuracy on the test data and was cross-validated using 5 folds.
Bayesian inference:
Using a quality score of 6 or higher to define "good" wines, this method explored the link between physiochemical properties and the probability of a wine being "good". Wines with citric acid levels between 0.29 and 0.34 g.dm-3 showed the highest probability.
