### Project Overview

Ligand-X serves as a bridge between complex computational chemistry tools and researchers who may not have years of experience in the field. It provides a suite of microservices for cheminformatics/bioinformatics simulation, all accessible through a modern, intuitive web interface. It is powered by open source Python tools and toolkits in the backend, and has a modular setup, so you only have to download what you want to use.

### Technical Stack

-   **Frontend**: React, Next.js, Tailwind SCSS
-   **Backend**: Python Microservices
-   **Chemistry Tools**: RDKit, OpenMM, AutoDock Vina, OpenFF, Psi4
-   **Infrastructure**: Docker, Kubernetes

### Workflow Examples

The application is designed to be modular. You can run a full pipeline from cleaning to free energy calculation, or just use individual tools as needed.
